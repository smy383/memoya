🔒개인정보처리방침

메모야(Memoya)는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 방침은 앱에서 수집하는 정보와 그 사용 방법에 대해 설명합니다.

1. 수집하는 개인정보

메모야는 다음과 같은 정보를 수집합니다:

  • 메모 내용: 사용자가 작성한 텍스트 메모
  • AI 채팅 기록: AI 어시스턴트와의 대화 내용
  • 앱 설정 정보: 언어 설정, 테마(다크/라이트 모드) 선택
  • 앱 사용 기록: 메모 작성 시간, 채팅 사용 패턴

2. 개인정보 저장 방식

🏠완전 로컬 저장: 모든 개인정보는 사용자의 기기에만 저장됩니다.

  • 메모와 채팅 기록은 기기의 AsyncStorage에 저장
  • 서버로 전송되지 않음 (AI API 사용 시 제외)
  • 앱 삭제 시 모든 데이터가 완전히 제거됨

3. AI 서비스 이용 시 정보 처리

AI 채팅 기능 사용 시:

  • Google Gemini API를 통해 대화 내용이 일시적으로 전송됩니다
  • 프록시 서버를 통해 API 키를 안전하게 관리합니다
  • 대화 종료 후 서버에서 자동으로 삭제됩니다
  • 메모 검색 및 요약을 위해 기기 내 메모 데이터를 활용합니다

4. 개인정보 이용 목적

  • 메모 작성, 수정, 삭제 기능 제공
  • AI 기반 메모 검색 및 요약 서비스 제공
  • 사용자 맞춤 설정 저장 (언어, 테마)
  • 앱 기능 개선을 위한 사용 패턴 분석

5. 개인정보 보유 및 이용 기간

  • 메모 및 채팅 데이터: 사용자가 직접 삭제하거나 앱을 제거할 때까지
  • 앱 설정 정보: 앱 사용 기간 동안
  • AI API 전송 데이터: 즉시 삭제 (서버에 저장되지 않음)

6. 개인정보의 제3자 제공

🚫제3자 제공 없음: 메모야는 사용자의 개인정보를 제3자에게 제공하지 않습니다.

예외: Google Gemini API 사용 시에만 대화 내용이 일시적으로 전송되며, 이는 서비스 제공을 위한 필수적 처리입니다.

7. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:

  • 열람권: 앱 내에서 저장된 모든 메모와 채팅 기록 확인 가능
  • 수정·삭제권: 개별 메모와 채팅 메시지 수정 및 삭제 가능
  • 처리정지권: 앱 삭제를 통해 모든 개인정보 처리 중단 가능
  • 데이터 백업/복구: 파일 내보내기/가져오기 기능 제공

8. 개인정보 보호를 위한 기술적 대책

  • 로컬 저장소(AsyncStorage) 암호화
  • API 통신 시 HTTPS 보안 연결
  • 프록시 서버를 통한 API 키 보안 관리
  • 앱 코드 난독화 (ProGuard 적용)

9. 개인정보처리방침의 변경

본 개인정보처리방침은 관련 법령 및 정책 변경에 따라 수정될 수 있으며, 변경 시 앱 업데이트를 통해 공지합니다.
